## Who I am

I'm a frontend developer based in Nepal who cares about the craft — the spacing, the transitions, the interaction rhythm, and all the decisions most people never notice but always feel. I work primarily in **React**, **Next.js**, **TypeScript**, and **Tailwind CSS**, and I complement that with backend work in **Python** and **Django** when the project calls for it.

My work tends to sit somewhere between utility and personality. I'm drawn to interfaces that have a point of view — things that feel designed rather than assembled. Lately I've been thinking a lot about retro-inspired aesthetics, AI-assisted creative tools, and what it means to build a frontend with real architectural intent.

I have **25 public repositories** across frontend, fullstack, and applied ML. I hold the **Pull Shark**, **YOLO**, and **Quickdraw** GitHub achievements.

---

## What I'm working on

- Pushing toward intentional, well-documented design systems
- Experimenting with retro UI aesthetics and visual character
- Deepening fullstack architecture through Python & Django
- Exploring AI-assisted creative and development workflows
- Building things that have both function and personality
